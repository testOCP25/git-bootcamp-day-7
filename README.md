# taskmanager

Учебный CLI-менеджер задач на Python. Позволяет добавлять задачи, просматривать список и помечать их как выполненные.

Минимальный API в `tasks.py`: `add_task(title)`, `list_tasks()`.

## Функциональность

- Добавление задач с валидацией непустого названия;
- Просмотр списка задач;
- Завершение задачи по индексу;
- Удаление задачи по индексу.


---

## Установка

```bash
git clone https://github.com/slurm/tasks.git
cd tasks/
```

---

## Запуск

```bash
python3 -c "import tasks; tasks.add_task('Прибить гвоздь');tasks.add_task('Закрутить шуруп');tasks.add_task('Забить саморез');tasks.complete_task(2); tasks.list_tasks()"
```
